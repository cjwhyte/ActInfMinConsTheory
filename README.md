# On the Minimal Theory of Consciousness Implicit in Active Inference

Github repository containing the MATLAB scripts necessary to reproduce the simulations reported in:

https://www.sciencedirect.com/science/article/pii/S1571064525001617?via%3Dihub
https://arxiv.org/abs/2410.06633

All simulations are run in SPM 12 using the DEM toolbox. Model inversion script is included in repository for reproducibility as SPM is updated.
